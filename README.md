# 我的导航页

这是一个面向全栈开发工程师的本地导航页。

## 当前结构

- `index.html`：页面结构与样式
- `config.js`：个人信息、主题、网站分组配置
- `app.js`：页面渲染和交互逻辑
- `nav-home-redirect.user.js`：历史脚本（当前不作为主方案）

## 已完成

1. 页面恢复个性化信息（头像、签名、标签、时间、主题切换）
2. 配置和逻辑拆分为独立 JS
3. 增加常用全栈站点（包含 Spring 官网、IP 检测、IT 工具箱、ping0、ip111 等）
4. 文件统一为 UTF-8 编码

## 本地使用

直接打开 `index.html`。
修改链接请编辑 `config.js` 中的 `linkGroups`。
